Here's how to install Rust and Cargo on WSL:
## Step 1: Open WSL

```bash
wsl
```
## Step 2: Update your system

```bash
sudo apt update && sudo apt upgrade -y
```

## Step 3: Install Rust using rustup

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

When prompted, choose option `1` (default installation).
## Step 4: Configure your shell

```bash
source "$HOME/.cargo/env"
```

Or restart your terminal.

## Step 5: Verify installation

```bash
rustc --version
cargo --version
```
You need to install the C compiler and build tools. Run this command:
```bash
sudo apt install build-essential
```
This will install:
- `gcc` (C compiler)
- `g++` (C++ compiler)
- `make` and other build tools
